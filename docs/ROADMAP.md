# RYVEN Webstore Roadmap

## Build Sequence

1. Authentication system (done)
2. Product listing and product details (done)
3. Cart and cart persistence
4. Checkout with Razorpay and signature verification
5. Order system and webhook-driven payment status
6. Customer dashboard
7. Admin dashboard with full management features

## Rules

- Keep `frontend` and `backend` fully independent.
- Follow modular architecture (no mega files).
- Security first in every feature.
- Optimize for real mobile users on VPS deployment.
